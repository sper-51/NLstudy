# 智能在线考试平台 - 联调方案

## 一、项目概述

本项目是前后端分离的在线考试系统：
- **后端**：SpringBoot 3.2.0 + MyBatisPlus 3.5.5 + MySQL 8.0
- **前端**：Vue.js + Vite + Element Plus
- **端口**：后端 8080，前端 3000

## 六、常见问题

### 6.1 端口被占用

```bash
# Windows 查看端口占用
netstat -ano | Select-String ":8080"
netstat -ano | Select-String ":3000"
```

### 6.2 数据库连接失败

- 确认 MySQL 服务已启动
- 确认端口映射正确（3306 容器端口 → 3307 宿主机端口）
- 确认 application.yml 中的端口与实际一致

### 6.3 MySQL 保留字问题

`rank` 是 MySQL 保留字，在实体字段上使用 `@TableField("`rank`")` 注解。

### 6.4 前端无法访问后端

- 检查后端是否启动在 8080 端口
- 检查 vite.config.js 中的代理配置

### 6.5 教师端页面显示空数据

- `teacher/grading/tasks` 返回空 → 数据库中无待批改的主观题答卷
- `teacher/grading/reviews` 返回空 → 数据库中无复核申请记录
- `teacher/exams/{id}/monitor` 返回空 → 后端未实现实时监控数据
