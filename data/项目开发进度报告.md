## 一、项目概述

本项目是一个基于 Spring Boot + Vue.js 的智能在线考试平台，包含**学生端、教师端、管理端**三个角色模块。

### 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 后端框架 | Spring Boot | 3.2.0 |
| ORM | MyBatis Plus | 3.5.5 |
| JWT | JJWT | 0.12.5 |
| 安全 | Spring Security | (已集成) |
| 前端框架 | Vue.js | 3.x |
| 构建工具 | Vite | 6.x |
| 语言 | TypeScript | 5.x |
| UI组件库 | Element Plus | 最新 |
| 状态管理 | Pinia | - |
| 路由 | Vue Router | 4.x |
| HTTP客户端 | Axios | - |
| 数据库 | MySQL | 8.0+ (Docker) |
| 图表 | ECharts | - |

## 六、关键技术说明

### 认证流程
```
前端 POST /api/v1/auth/login {username, password}
  → AuthController.login()
  → 验证用户名密码(BCrypt匹配)
  → 生成JWT Token(JwtUtil, 24h有效期)
  → 返回 {token, userInfo(id, username, real_name, role)}
  → 前端存入 localStorage + Pinia store
  → 根据role自动redirect: student→/home, teacher→/teacher/dashboard, admin→/admin
```

### 请求链路
```
前端 Axios(request.ts baseURL=/api/v1)
  → 拦截器自动附加 Authorization: Bearer {token}
  → Vite Proxy → localhost:8080/api/v1/...
  → JwtInterceptor.preHandle() 验证token
  → 解析userId写入 request.setAttribute("X-User-Id")
  → Controller 通过 @RequestHeader("X-User-Id") 获取当前用户ID
  → 业务处理 → 返回 Result<T> 统一响应格式
```

### 统一响应格式
```json
{
  "code": 200,
  "message": "success",
  "data": { ... }
}
```

### 字符编码
- **数据库连接**: `charset=utf8mb4` (必须！否则中文乱码)
- **后端Filter**: `CharacterEncodingFilter` (UTF-8 Request/Response)
- **WebConfig**: `StringHttpMessageConverter` 默认UTF-8
- **数据插入**: 通过 Python pymysql 以 utf8mb4 charset 连接执行SQL
