## 1. 文档概述

### 1.1 产品背景

随着教育信息化的深入发展，传统的线下考试模式面临诸多挑战：人工组卷耗时长、纸质试卷管理成本高、人工阅卷效率低且主观性强、考试数据分析困难等。为解决上述痛点，特规划建设智能在线考试平台。

本平台旨在为教育机构提供一套完整的在线考试解决方案，支持多种题型、智能组卷、自动评分、数据分析等功能，提高考试效率，减少人工干预，降低运营成本。

### 1.2 产品定位

面向教育机构（中小学校、培训机构、职业院校等）提供轻量级在线考试服务，聚焦核心考试功能，支持师生日常测验、阶段考核、模拟练习等场景。

### 1.3 目标用户

| 角色 | 说明 | 使用场景 |
|------|------|----------|
| 学生 | 参加在线考试、完成课后练习 | 参加考试、查看成绩、错题复习 |
| 教师 | 出题、组卷、阅卷、分析成绩 | 创建试卷、安排考试、批改主观题、成绩分析 |
| 管理员 | 系统运营监控与数据分析 | 数据看板、系统监控、配置管理 |

### 2.2 学生端功能

- 学生通过账号密码登录系统
- 首页展示个人信息和待参加考试列表
- 显示近期考试成绩和排名
- 显示所在班级信息

#### 2.2.2 我的班级

| 功能 | 说明 |
|------|------|
| 加入班级 | 输入班级码申请加入老师创建的班级 |
| 班级信息 | 查看班级名称、老师、成员列表 |
| 班级公告 | 查看老师发布的班级通知 |

#### 2.2.3 错题复习

| 功能 | 说明 |
|------|------|
| 错题列表 | 查看历史考试中的所有错题 |
| 错题分类 | 按知识点/题型/考试筛选错题 |
| 错题详情 | 查看题目、正确答案、自己的答案、解析 |
| 练习模式 | 重新作答错题，检验是否掌握 |
| 错题重做记录 | 记录每次重做的得分变化 |

| 功能 | 说明 |
|------|------|
| 参加考试 | 查看待考任务，点击进入考试界面 |
| 考试计时 | 考试界面实时显示剩余时间 |
| 答题功能 | 支持单选、多选、判断、填空、简答等题型 |
| 答题进度 | 显示已答/未答/标记题目数量 |
| 切题功能 | 点击题号快速切换题目 |

- 查看历史考试成绩

- 查看每道题的答题情况（正确/错误）

- 修改登录密码
- 查看个人信息
- 退出班级

| 功能 | 说明 |

| 填空题 | 4 | 一个或多个空 | 支持 |

### 3.2 闭环一：考试发布与参加流程

#### 流程步骤

```
教师                          系统                          学生
  │                             │                            │
  │──1.1 创建班级──────────────▶│                            │
  │                             │──1.2 生成班级码───────────▶│
  │                             │                            │
  │──2.1 添加班级题库──────────▶│                            │
  │                             │                            │
  │──3.1 创建试卷──────────────▶│                            │
  │     (选择题目,设置分值)     │                            │
  │                             │                            │
```

#### 详细说明

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1.1 | 创建班级 | 教师创建班级，系统生成唯一班级码 |

| 4.1 | 安排考试 | 选择试卷、班级、时间范围，设置考试规则 |
| 4.2 | 推送通知 | 系统向班级学生推送考试任务 |
| 4.3 | 查看待考 | 学生可在首页/考试中心看到待参加考试 |

### 3.3 闭环二：在线答题与自动评分流程

#### 流程步骤

```
学生                                                    系统
  │                                                      │
  │──1. 进入考试───────────────────────────────▶│──1.1 验证考试资格
  │                                           │──1.2 加载试卷题目
  │◀──2. 展示试卷界面（计时器/题目列表）───────│
  │                                                      │
  │──3. 选择/填写答案───────────────────────▶│──3.1 实时保存答案
  │     (单选/多选/判断/填空/简答)           │     (每题自动保存)
  │                                                      │
  │──4. 切换题目（上一题/下一题/题号跳转）───▶│
  │                                                      │
  │──5. 标记不确定题目─────────────────────▶│
  │                                                      │
  │──6. 确认交卷───────────────────────────▶│──6.1 自动评分
  │                                           │     (客观题即时评分)
  │                                           │──6.2 标记主观题待阅卷
  │◀──7. 展示客观题得分/提交成功──────────────│
  │                                                      │
```

#### 详细说明

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 进入考试 | 学生点击"开始考试"，系统验证考试资格（时间范围内、未参加过） |
| 2 | 展示试卷 | 加载试卷，显示考试说明、题型、计时器 |
| 3 | 答题 | 选择/填写答案，每题答案自动即时保存到服务器 |
| 4 | 切题 | 支持上一题/下一题/题号快速跳转 |
| 5 | 标记 | 可标记不确定题目，便于回头检查 |
| 6 | 交卷 | 学生确认交卷后，系统自动评分（客观题即时，主观题标记待阅卷） |
| 7 | 结果展示 | 展示已评分得分，主观题显示"待阅卷"状态 |

### 3.4 闭环三：主观题阅卷与成绩发布流程

#### 流程步骤

```
教师                          系统                          学生
  │                             │                            │
  │──1. 查看待阅卷─────────────▶│                            │
  │                             │──1.1 获取待阅卷列表────────▶│
  │◀──2. 待阅卷列表─────────────│                            │
  │     (含主观题的考试)         │                            │
  │                             │                            │
  │──3. 批改主观题──────────────▶│──3.1 保存得分───────────▶│
  │     (查看学生答案,给分)      │                            │
  │                             │──3.2 更新总分───────────▶│
  │                             │──3.3 判定考试完成────────▶│
  │                             │                            │
  │──4. 成绩发布───────────────▶│──4.1 推送成绩通知────────▶│
  │     (手动或自动)             │                            │──4.2 收到成绩提醒
  │                             │                            │
```

#### 详细说明

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 查看待阅卷 | 教师进入成绩管理，看到包含主观题且有待批改的试卷列表 |
| 3 | 批改主观题 | 教师查看学生作答内容，输入得分 |
| 3.1 | 保存得分 | 每题批改后即时保存 |
| 3.2 | 更新总分 | 客观题得分+主观题得分=总分 |
| 3.3 | 判定完成 | 主观题全部批改完毕后，考试状态变为"已完成" |
| 4 | 成绩发布 | 可手动发布，也可设置为主观题批改完自动发布 |
| 4.1 | 推送通知 | 系统向学生推送成绩可查通知 |
| 4.2 | 收到提醒 | 学生收到消息，可查看个人成绩 |

### 3.5 闭环四：成绩查看与错题复习流程

#### 流程步骤

```
学生                                                    系统
  │                                                      │
  │──1. 查看成绩列表──────────────────────────────────▶│──1.1 获取成绩记录
  │◀──2. 成绩列表（考试名/得分/排名/时间）──────────────│
  │                                                      │
  │──3. 查看成绩详情──────────────────────────────────▶│──3.1 获取答题详情
  │◀──3.2 成绩详情─────────────────────────│            │
  │     (每题得分/正确答案/学生答案)        │            │
  │     (标记错题)                          │            │
  │                                                      │
  │──4. 进入错题本───────────────────────────────────▶│──4.1 获取错题列表
  │◀──4.2 错题列表───────────────────────────│           │
  │     (按知识点/题型分类)                   │           │
  │                                                      │
  │──5. 错题练习────────────────────────────────────▶│──5.1 加载错题
  │◀──5.2 重新作答───────────────────────────│          │
  │                                                      │
  │──6. 提交练习答案───────────────────────────────▶│──6.1 判定对错
  │◀──6.2 展示结果/更新掌握状态──────────────────│          │
  │                                                      │
```

#### 详细说明

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 查看成绩列表 | 学生进入成绩查询，看到所有历史考试成绩 |
| 3 | 查看详情 | 点击考试查看每题得分，可标记需要复习的错题 |
| 4 | 进入错题本 | 错题自动收录，可按知识点/题型/考试筛选 |
| 5 | 错题练习 | 选择题目重新作答，检验是否已掌握 |
| 6 | 提交练习 | 系统判定对错，记录重做结果，连续做对标记为"已掌握" |

#### 详细说明

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 班级概览 | 查看班级整体情况：学生数量、考试次数、平均分 |
| 3 | 成绩分析 | 查看班级成绩分布（柱状图/饼图）、及格率、学生排名 |
| 4 | 题目分析 | 统计各题目正确率，发现高频错题和薄弱知识点 |
| 5 | 调整策略 | 根据数据分析结果，制定针对性教学计划 |
| 6 | 布置练习 | 基于薄弱知识点创建专项练习试卷，推送给学生 |

```
                                      │            ┌──────────┐  │
```

| 状态 | 说明 | 触发条件 |
|------|------|----------|
| 已发布 | 考试已安排，学生可查看 | 教师发布考试 |
| 进行中 | 学生正在答题 | 学生在考试时间内开始答题 |
| 已提交 | 学生已交卷 | 学生点击交卷 |
| 待阅卷 | 等待教师批改主观题 | 交卷后存在未评分主观题 |
| 已完成 | 全部评分完成 | 所有题目评分完毕 |
